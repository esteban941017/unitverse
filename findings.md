# Project Findings

## Research
- The application will be a static web app hosted on AWS S3.
- Requires Domain-Driven Design (DDD) structure in a React/TypeScript context.

## Discoveries
- **North Star:** A responsive webpage that converts different kinds of units, with ad service integrations for monetization.
- **Integrations:** AWS S3 for hosting. Ad Services for monetization.
- **Tech Stack:** React, TypeScript, Node.js (for tooling/build).
- **Architecture:** Domain Driven Design (DDD) pattern.
- **Delivery Payload:** Production build files (HTML, CSS, JS) optimized for S3 static website hosting.

## Constraints
- Must be stateless or rely on client-side logic primarily, given the S3 hosting constraint.
